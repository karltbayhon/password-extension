# Password Strength Visualizer Chrome Extension

## Project Description
This project is a simple Chrome extension that checks the strength of a password and provides instant feedback.

## Features
- Detects weak, medium, and strong passwords
- Checks password length
- Detects uppercase letters, numbers, and symbols
- Simple and easy-to-use interface

## How to Install

1. Download the project folder
2. Open Google Chrome
3. Go to chrome://extensions/
4. Enable Developer Mode
5. Click "Load unpacked"
6. Select the project folder

## Technologies Used
- HTML
- CSS
- JavaScript
- Chrome Extension API

## Purpose
The extension helps users quickly evaluate password strength to encourage better online security practices.
